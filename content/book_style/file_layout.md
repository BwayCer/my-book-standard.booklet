```md
## 常見問題是否有提供書籤連結？


有的。你能透過下述方式連結至此問題項目：

  * `網址#<編號>`。 如： [/appendix/faq.html#1](#1)
  * `網址#<標題>`。 如： [/appendix/faq.html#常見問題是否有提供書籤連結？](#常見問題是否有提供書籤連結？)

```
